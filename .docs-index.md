## Environment Setup
Quick reference:
1. Install: `pnpm install`
2. Env: Copy `.env.example` → `.env.local`, fill in values
3. DB: `pnpm db:generate && pnpm db:migrate`
4. Run: `pnpm dev`
5. Test: Visit http://localhost:3000

Full details: README.md "Getting Started"
